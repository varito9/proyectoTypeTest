## CONCEPTE DEL JOC

<p>El nostre Type Racer Royale  es una batalla de mecanografia màgic en la que diversos jugadors s'uneixen a una sala. Quan comença la partida, a tots els apareix una pila de paraules. A mesura que completen paraules correctamet obtenen powerups que poden penalitzar als seus oponents, fent que les seves piles creixin ,desapareguin o alentir-se. Si els errors de paraules d'un jugador arriba a un límit màxim, queda eliminat.</p>
